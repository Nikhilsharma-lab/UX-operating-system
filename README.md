# Nikhil Sharma Executive Knowledge System

This repository contains the strategy, content, design, implementation, and operating system for Nikhil Sharma’s executive knowledge platform.

This is not a traditional UX portfolio.

It is a long-term knowledge system documenting how AI-native product organizations are built.

## Core Thesis

Building AI-native product organizations that transform ambiguity into measurable business outcomes.

## Positioning

Nikhil Sharma is positioned as an AI-native Product Executive who builds organizations, systems, and businesses.

## The Website Is

- An executive knowledge system
- A digital headquarters
- A transformation archive
- A design leadership operating manual
- A writing and thinking system
- A proof system for senior leadership roles
- A long-term career asset

## The Website Is Not

- A Behance portfolio
- A Dribbble gallery
- A resume website
- A generic UX case study collection
- A visual design showcase without thinking
- A blog of disconnected opinions

## Build Philosophy

Every page must answer one executive hiring question.

Every page must either:

- Teach
- Prove
- Clarify
- Demonstrate judgment
- Demonstrate capability
- Demonstrate business impact

If it does not, it should not exist.

## Core Files

- `PROGRAM_INDEX.md` — the master dashboard: synthesis, operating model, the route, the map, quality systems, decision log, parking lot, content graph.
- `MASTER_PLAYBOOK.md` — the canonical strategy and content document.
- `AI_COLLABORATION_MODEL.md` — how AI tools are used to build the system.
- `DESIGN.md` — the root-level design contract Claude Code must read before any UI change.
- `DECISION_LOG.md` — the running record of decisions.
- `ANTI_GOALS.md` — what the system must never become.
- `VOCABULARY.md` — the preferred language of the system.
- `CONTENT_GRAPH.md` — how ideas connect across the system.

## AI Collaboration Model

- Nikhil owns final judgment.
- ChatGPT owns strategy, writing, critique, and prompts.
- Claude Code owns implementation.
- Claude Chat may be used only for technical second opinions.
