# RELEASES/V1.md

## Purpose

This file defines the first public launch of the Nikhil Sharma Executive Knowledge System.

V1 must be strong, focused, and live.

V1 is not the full vision.

V1 is the first complete vertical slice.

---

## V1 Goal

Create a live executive website that passes two tests:

1. Recruiter understands Nikhil in 30 seconds.
2. VP of Design stays for 5 minutes and sees enough evidence to want a conversation.

---

## V1 Scope

### Required

- Home
- Capabilities
- Transformations Index
- Three Transformation Stories
- Contact
- Navigation
- Footer
- Basic SEO
- Basic Analytics
- Accessibility baseline
- Responsive design

### Preferred

- Thinking index with 1–2 seeded essays
- Operating Manual preview
- Builder preview

### Not Required

- Full Operating Manual
- Full essay library
- Full framework library
- Newsletter
- YouTube
- Podcast
- Community
- Courses
- Book

---

## V1 Transformation Stories

Required:

1. Building the Design Organization
2. AI-native Product Development
3. Digital Gold Growth or Fraud Prevention

Pick the third story based on evidence strength.

---

## V1 Definition of Done

- [ ] Homepage passes 30-second recruiter test
- [ ] Site has at least three strong transformation stories
- [ ] Navigation is clear
- [ ] Claims are defensible
- [ ] Works on mobile and desktop
- [ ] Lighthouse score 95+
- [ ] Basic SEO complete
- [ ] Analytics wired
- [ ] Contact path clear
- [ ] No generic portfolio language
- [ ] No unsupported claims
