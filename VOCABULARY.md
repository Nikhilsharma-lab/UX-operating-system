# VOCABULARY.md

## Purpose

This file defines the preferred language for the Nikhil Sharma Executive Knowledge System.

Consistent vocabulary creates a consistent brand.

---

## Preferred Terms

Use:

- Executive Knowledge System
- Operating Manual
- Transformation Story
- Capability
- Product Organization
- AI-native Product Development
- Product Infrastructure
- Organizational Design
- Behavioral Growth
- Customer Trust
- Systems Thinking
- Decision Quality
- Operating Model
- Evidence
- Tradeoff
- Outcome
- Framework
- Playbook
- Knowledge Graph
- Transformation Archive

---

## Avoid

Avoid:

- Portfolio
- Case Study
- Project
- UX Portfolio
- Screens
- Pixel-perfect
- Passionate
- Visionary
- World-class
- Cutting-edge
- Innovative
- Design guru
- Thought leader
- Rockstar
- Ninja
- UI showcase
- Dribbble-style
- Behance-style
