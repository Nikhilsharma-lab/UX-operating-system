## The Six Quality Gates

### Executive Review

- [ ] Does this reinforce Nikhil as an AI-native product executive?

### Story Review

- [ ] Does it communicate a transformation, not just a project?

### Craft Review

- [ ] Is the writing, layout, spacing, typography, and visual rhythm excellent?

### Evidence Review

- [ ] Are claims backed by outcomes, metrics, decisions, or proof?

### Evergreen Review

- [ ] Will this still feel useful and credible five years from now?

### Compounding Review

- [ ] Can this idea become an essay, talk, LinkedIn post, video, framework, or course module?

## Design Consistency Review

- [ ] Claude Code read `DESIGN.md`
- [ ] Typography follows approved hierarchy
- [ ] Spacing follows system rhythm
- [ ] Motion is purposeful
- [ ] Color usage is restrained
- [ ] Components feel systemic
- [ ] No generic portfolio patterns introduced
- [ ] No decorative UI added without purpose
- [ ] No shadows introduced
- [ ] No gradients introduced
- [ ] Highlighter yellow appears only as restrained punctuation
